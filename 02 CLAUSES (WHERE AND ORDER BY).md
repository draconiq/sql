#### CLAUSES (WHERE AND ORDER BY)



##### WHERE - FILTERS THE DATA ON THE CONDITION 



###### 

###### SELECT \* FROM TABLE\_NAME 

###### WHERE CONDITION 

###### 

###### WHERE CONDITION CHARACTERS MUST HAVE SINGLE QUOTES ' '

##### 

##### ORDER BY - SORTING THE DATA 

###### 

###### 1\. ASCENDING LOW TO HIGH  - ASC

###### 2\. DESCENDING HIGH TO LOW - DESC



###### SYNTAX -

###### 

###### SELECT \* FROM TABLE\_NAME 

###### ORDER BY COL\_NAME ASC

###### 

###### IF NOT SPECIFIED ASC OR DESC THE DEFAULT WILL BE ASCENDING 



##### NESTED ORDER BY



###### (NESTED SORTING)

###### MULTIPLE COULUMN 

###### 

###### SYNTAX -

###### 

###### SELECT \* FROM TABLE\_NAME 

###### ORDER BY COL1 CONDITION, COL2 CONDITION, COL3 CONDITION
