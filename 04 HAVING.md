###### HAVING:



###### FILTERS THE DATA AFTER THE AAGGREGATION (GROUP BY), 

###### CAN ONLY BE USED WITH GROUP BY 



##### SYNTAX -



###### SELECT COUNTRY, SUM(SCORE)

###### FROM TABLE\_NAME

###### GROUP BY COUNTRY

###### HAVING SUM(SCORE)>800



###### HAVING CONDTION



##### WHERE COMES BETWEEN FROM AND GROUP BY 

##### 

##### SYNTAX -



###### SELECT COUNTRY, SUM(SCORE)

###### FROM TABLE\_NAME

###### WHERE SCORE>400

###### GROUP BY COUNTRY

###### HAVING SUM(SCORE)>800

##### FIND AVG SCORE OF EACH COUNTRY

##### CONSIDERING ONLY CUSTOMERS WITH A SCORE NOT EQUAL TO 0

##### AND RETURN ONLY THOSE COUNTRIES WITH AVG SCORE GREATER THAN 430



###### SYNTAX - 

###### 

###### SELECT COUNTRY, AVG(SCORE) AS AVG\_SCORE

###### FROM CUTOMERS

###### WHERE SCORE != 0

###### GROUP BY COUNTRY
