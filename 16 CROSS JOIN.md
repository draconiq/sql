#### CROSS JOIN 



##### COMBINES EVERY ROW FROM RIGHT WITH EVERY ROW FROM THE LEFT 

##### ALL POSSIBLE COMBINATIONS  ALSO CALLED AS CARTESIAN JOIN 

##### 

##### IF A COLUMN AS 2 ROWS AND B HAS 3 ROWS THIS GENERATES 6 POSSIBLE COMBINATIONS



#### SYNTAX - 



###### SELECT \* FROM TABLE\_NAME  

###### CROSS JOIN COLUMN\_NAME
