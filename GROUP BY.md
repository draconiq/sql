#### **GROUP BY** 



THIS COMES BETWEEN WHERE AND ORDER BY



COMBINES ROWS WITH THE SAME VALUE

AGGREGATES A COLUMN BY ANOTHER COLUMN



TOTAL SCORE BY 1 COUNTRY (CATEGORY)



###### **SYNTAX -** 



SELECT COUNTRY, SUM(SCORE) 

FROM TABLE\_NAME GROUP BY COUNTRY



SUM(SCORE) ADDS THE VALUE  OF MULTIPLE COLUMN WITH SAME NAME



AFTER EXECUTING THE NEW COLUMN WILL HAVE NO NAME SO

WE CAN GIVE ALIAS FOR THE COLUMN LIKE 

SUM(SCORE) AS TOTAL\_SCORE

###### 

###### NOW SYNTAX WILL BE AS 



SELECT COUNTRY, SUM(SCORE) AS TOTAL\_SCORE

FROM TABLE\_NAME GROUP BY COUNTRY



THAT COLUMN ONLY EXIST IN THIS QUERY IT CANNOT BEE USED ANYWHERE ELSE 

NOT SAVED IN OTHER DATABASE AS WELL



RESULT OF GROUP BY DETERMINED BY THE UNIQUE

VALUES OF THE GROUPED COLUMNS.



EX: SELECT COUNTRY, FIRST\_NAME , SUM(SCORE) AS TOTAL\_SCORE

&#x20;   FROM TABLE\_NAME GROUP BY COUNTRY

THIS GIVES AN ERROR CAUSE FIRST NAME IS NOT AGGREGATTED OR MENTIONED IN GROUP BY,



FOR FIRST\_NAME TO GET MENTIONED WITHOUT AN ERROR 



SYNTAX- 



SELECT COUNTRY, FIRST\_NAME, SUM(SCORE) AS TOTAL\_SCORE

FROM TABLE\_NAME GROUP BY COUNTRY, FIRST\_NAME
